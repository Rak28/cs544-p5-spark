# CS544 P5: Spark

**Course:** CS544 — Big Data Systems, UW-Madison (Fall 2025)

## Overview

Used **Apache Spark** and **Hive** to analyze a dataset of competitive programming problems and solutions. Loaded data into Hive tables/views and performed complex joins and aggregations to answer analytical questions.

## Learning Objectives

- Load and query data using Apache Spark and Hive
- Perform multi-table joins to resolve IDs to meaningful values
- Use PySpark DataFrame API for distributed data processing
- Analyze large datasets efficiently with columnar storage
